# Social Content Routine Pack

Social content routines cover LinkedIn, Medium, X/Twitter, Instagram, Facebook,
YouTube, and TikTok surfaces for read-only, profile review, notification review,
draft, and explicitly approved publish flows. Publish-capable routines must
carry approval and checkpoint metadata before promotion.

## Seed Matrix

Each platform has read-only, profile review, notification review, draft, and
approved-publish seed routines:

- Instagram
