# Browser Routine Pack

Browser routines cover navigation, tabs, search, reading, forms, downloads,
exports, settings, bookmarks, history, and browser-based writing surfaces.
Routines in this pack should prefer stable text/UIA/image selectors, include
allowed-window rules, and keep trace evidence local.
