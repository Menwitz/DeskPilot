# Browser Routine Pack

Browser routines cover navigation, search, reading, forms, downloads, settings,
and browser-based writing surfaces. Routines in this pack should prefer stable
text/UIA/image selectors, include allowed-window rules, and keep trace evidence
local.

## Seed Routines

- `browser.navigation-open-page`
- `browser.form-fill-basic`
- `browser.search-web`
- `browser.read-page`
- `browser.extract-visible-text`
- `browser.writing-surface-draft`
- `browser.download-open`
- `browser.settings-open`
